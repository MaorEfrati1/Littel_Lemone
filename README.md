## 📖 About the Project

**Little Lemon** is a mobile app for a family-owned Mediterranean restaurant based in Chicago. The restaurant is focused on traditional recipes served with a modern twist — and this app brings that experience to your fingertips.

This project was built as the **capstone project** for the [Meta React Native Specialization](https://www.coursera.org/specializations/meta-react-native) on Coursera, demonstrating real-world mobile development skills including navigation, state management, local storage, and UI design.

## ✨ Features

- 🧅 **Onboarding Screen** — Simple registration with first name and email
- 🍽️ **Menu Screen** — Browse all dishes with live search and category filters (All, Starters, Mains, Desserts)
- 👤 **Profile Screen** — Edit personal info, upload/remove avatar, and manage email notification preferences
- 💾 **Persistent Storage** — User data and preferences saved with AsyncStorage
- 🔍 **Real-time Search** — Filter menu items as you type
- 🏷️ **Category Filtering** — Toggle between food categories instantly
- 🔔 **Email Notifications** — Subscribe to order updates, offers, and newsletters

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- Expo CLI
- Android/iOS simulator or physical device with Expo Go

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/Little_Lemon.git

# 2. Navigate to the project folder
cd Little_Lemon

# 3. Install dependencies
npm install

# 4. Start the development server
npx expo start
```

Then scan the QR code with **Expo Go** on your phone, or press `a` for Android / `i` for iOS simulator.
